# User Payloads

## Create User (POST /users)
```json
{
  "username": "testuser",
  "email": "test@example.com",
  "password": "password"
}
```

## Create User - Missing Fields (POST /users)
```json
{
  "username": "testuser"
}
```

## Update User (PUT /users/{user_id})
```json
{
  "username": "newusername"
}
```

# Post Payloads

## Create Post (POST /posts)
```json
{
  "title": "Test Post",
  "content": "This is a test post."
}
```

## Create Post - Missing Title (POST /posts)
```json
{
  "content": "This is a test post."
}
```

## Create Post - Missing Content (POST /posts)
```json
{
  "title": "Test Post"
}
```

## Create Post - Special Characters in Title (POST /posts)
```json
{
  "title": "Test Post!",
  "content": "This is a test post."
}
```

## Create Post - No Data (POST /posts)
```json
{}
```

## Update Post (PUT /posts/{post_id})
```json
{
  "title": "New Title"
}
```
